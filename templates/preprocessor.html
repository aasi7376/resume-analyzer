Module 03

NLP Preprocessing

This module uses spaCy to clean and normalize resume text before comparison

⚙️ How This Module Works
🔡
Step 1 — Lowercasing

All text is converted to lowercase for uniform comparison

✂️
Step 2 — Tokenization

Text is split into individual word tokens using spaCy

🚫
Step 3 — Stop Word Removal

Common words like "the", "is", "and" are removed from the text

🌱
Step 4 — Lemmatization

Words are reduced to their root form eg. "running" becomes "run"

💻 Module Code — preprocessor.py
import spacy
nlp = spacy.load("en_core_web_sm")

def preprocess_text(text):
    doc = nlp(text.lower())
    tokens = [
        token.lemma_
        for token in doc
        if not token.is_stop
        and not token.is_punct
        and not token.is_space
        and token.is_alpha
    ]
    return " ".join(tokens)

Module 04

Keyword Matching

This module identifies matched and missing keywords between your resume and the job description

⚙️ How This Module Works
🔑
Step 1 — Keyword Extraction

Important keywords are extracted from both resume and job description

🔍
Step 2 — Intersection

Keywords present in both resume and job description are identified as matched

❌
Step 3 — Difference

Keywords in job description but missing from resume are flagged as missing

📊
Step 4 — Match Ratio

Keyword match percentage is calculated as matched / total JD keywords × 100

💻 Module Code — matcher.py
def match_keywords(resume_text, jd_text):
    resume_keywords = extract_keywords(resume_text)
    jd_keywords     = extract_keywords(jd_text)

    matched = resume_keywords.intersection(jd_keywords)
    missing = jd_keywords.difference(resume_keywords)

    match_ratio = round(
        (len(matched) / len(jd_keywords)) * 100, 2
    )
    return {
        "matched": sorted(list(matched)),
        "missing": sorted(list(missing)),
        "match_ratio": match_ratio
    }

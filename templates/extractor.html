Module 02

PDF Text Extraction

This module uses PyPDF2 to extract raw text from your uploaded resume PDF

⚙️ How This Module Works
📂
Step 1 — File Upload

Resume PDF is received and securely saved to the uploads folder

📖
Step 2 — Page Reading

PyPDF2 opens the file and loops through every page of the document

🧹
Step 3 — Text Cleaning

Extra whitespace, special characters and non-English symbols are removed

✅
Step 4 — Output

Clean raw text string is passed to the NLP Preprocessing module

💻 Module Code — extractor.py
import re
import PyPDF2

def extract_text_from_pdf(file_path):
    text = ""
    with open(file_path, "rb") as f:
        reader = PyPDF2.PdfReader(f)
        for page in reader.pages:
            page_text = page.extract_text()
            if page_text:
                text += page_text + "\n"

    text = re.sub(r"\s+", " ", text)
    text = re.sub(r"[^\x00-\x7F]+", " ", text)
    return text.strip()
